This collection of example prompts demonstrates effective prompt engineering techniques for common development tasks. Use these as templates, customize them for your specific needs, and learn from the patterns to create your own effective prompts.

## Authentication & User Management

### **User Registration System**
```
I want to build a user registration system for my SaaS application.

Requirements:
- Email and password registration
- Email verification with confirmation link
- Password strength requirements (minimum 8 characters, uppercase, lowercase, number)
- Terms of service and privacy policy checkboxes
- Welcome email after successful verification
- Redirect to dashboard after login

Technical preferences: React with TypeScript, Tailwind CSS, Supabase for backend
Design: Clean, professional interface similar to Stripe's signup form

Please implement this step by step, starting with the registration form component.
```

### **Social Login Integration**
```
Add social login options to my existing authentication system.

Current setup: Email/password authentication with Supabase
New requirements:
- Google OAuth integration
- GitHub OAuth integration
- Maintain existing email/password option
- Single sign-on experience
- User profile creation from social data

Please show me how to configure the OAuth providers and update the UI to include social login buttons.
```

### **Password Reset Flow**
```
Implement a complete password reset flow for my app.

Requirements:
- "Forgot Password" link on login page
- Email input form for password reset request
- Secure reset token generation and validation
- New password input form with strength validation
- Success confirmation and automatic login
- Expire reset tokens after 1 hour

Use the existing Supabase setup and maintain the current design system.
```

```

Features needed:
- Sortable columns (name, email, role, status)
- Search/filter functionality
- Pagination with configurable page sizes
- Row selection with checkboxes
- Bulk actions (delete, export, change status)
- Responsive design for mobile devices

Data structure: Array of user objects with id, name, email, role, status, createdAt
Use React with TypeScript and implement proper accessibility features.
```
